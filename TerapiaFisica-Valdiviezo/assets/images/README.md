# 📁 Carpeta de Imágenes

## Instrucciones de uso

Coloca todas las imágenes en esta carpeta en formato **WebP** (.webp).

### Imagen requerida:

- **`hero.webp`** - Imagen de fondo para la sección Hero (se recomienda una imagen inspiradora de salud/fitness)
  - Dimensiones recomendadas: 1920x1080px o superior
  - Tamaño de archivo: 500KB - 2MB (WebP optimizado)

## Cómo convertir tus imágenes a WebP

### Opción 1: Usar herramientas online gratuitas
- https://cloudconvert.com/
- https://ezgif.com/
- https://www.pixelied.com/convert/webp

### Opción 2: Usar software local
- **ImageMagick** (Command line):
  ```
  magick convert imagen.jpg imagen.webp
  ```
- **XnConvert** (GUI gratuita)

### Opción 3: VSCode con extensiones
- Busca extensiones como "WebP Converter" en el marketplace de VSCode

## Notas importantes

- ✅ Todos los nombres de archivo deben coincidir exactamente (hero.webp)
- ✅ Usa WebP para mejor compresión y velocidad de carga
- ✅ Optimiza las imágenes para web
- ✅ Asegúrate de que las imágenes sean relevantes y profesionales
